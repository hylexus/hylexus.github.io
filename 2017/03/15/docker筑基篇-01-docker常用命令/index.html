1 docker 基础命令

info | version

ps

列出所有的container。不带参数的时候只是列出正在运行的container。

inspect

可以查看比ps更加底层的信息。

rm

删除指定容器

start | restart

启动/重启指定的容器

stop

停止一个正在运行的container

attach

可以连接到一个正在运行的container上。

logs

可以监控一个正在运行的容器的日志记录

top

exec

在一个正在运行的容器上执行命令

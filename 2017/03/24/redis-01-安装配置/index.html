2 mac安装

其实在mac下安装和在Linux(CentOS-6.8-x64)下是非常类似的。

在mac下安装redis最简单的方式就是使用brew install redis来一键搞定了。

如果你不喜欢瞎折腾，就是要最快的傻瓜式安装,那你就用上面的brew install redis吧。

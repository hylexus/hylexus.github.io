P01 (*) Find the last element of a list.

要求

找出list中的最后一个元素

方案

  • (1)List自带的last方法(废话)
  • (2)将list反转，之后取其head
  • (3)递归

P02(*) Find the last but one element of a list

要求

找出list中倒数第二个元素

Example:

方案

  • (1) reverse.tail.head
  • (2) init.last
  • (3) 递归

P03(*) Find the Kth element of a list.

要求

获取list中第n(从零开始)个元素

方案

  • (1) list自带索引(废话)
  • (2) 递归

P04 (*) Find the number of elements of a list.

要求

计算list的长度

Example:

方案

  • (1) list.length(废话)
  • (2) 普通递归
  • (3) 尾递归
  • (4) foldLeft
  • (5) 遍历list(没啥好说的)

P05 (*) Reverse a list.

要求

逆转一个list

Example:

方案

  • (1) list.reverse(废话)
  • (2) 递归
  • (3) 尾递归
  • (4) foldLeft
